# Security Policy

Security Consul is a security triage tool, so safe use matters.

## Responsible Use

Only scan websites you own, administer, or have explicit permission to test.
Do not use the project for unsolicited probing, spam, or abusive automation.

## Reporting Issues

If you find a vulnerability in this repository, please open a private GitHub
security advisory when available, or contact the maintainer through the GitHub
profile associated with this repository.

Please include:

- affected component
- reproduction steps
- expected impact
- suggested fix, if known

## Secrets

Never commit `.env`, API keys, Gmail app passwords, OAuth credentials, or local
state generated from real scans.
